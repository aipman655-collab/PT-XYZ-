Data Model Konseptual (ERD 6 Entitas)

1. users

user_id (PK)
name, email (UQ)
role, is_active

2. production_orders

order_id (PK)
product_name
target_qty, status, shift

3. machines

machine_id (PK)
machine_code (UQ)
status, location

4. production_records

record_id (PK)
order_id (FK), machine_id (FK)

5. quality_inspections

6. notifications

notif_id (PK)

Skema DDL Fisikal MySQL

Kod skema struktur DDL sedia dilaksanakan:

CREATE TABLE production_records (
  record_id INT AUTO_INCREMENT PRIMARY KEY,
  order_id INT NOT NULL,
  machine_id INT NOT NULL,
  shift ENUM('pagi','siang','malam'),
  actual_qty INT DEFAULT 0,
  defect_qty INT DEFAULT 0,
  recorded_by INT NOT NULL,
  FOREIGN KEY (order_id)
    REFERENCES production_orders(order_id),
  FOREIGN KEY (machine_id)
    REFERENCES machines(machine_id)
) ENGINE=InnoDB;
